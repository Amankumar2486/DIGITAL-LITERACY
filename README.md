
# Digital Literacy Project

Welcome to my **Digital Literacy** project!
This repository is created to spread awareness about **online safety, privacy, and responsible internet usage**, especially for students and beginners.

In today’s digital world, being smart online is just as important as being smart offline — and this project is a small step toward that goal 💡

---

## About the Project

This project focuses on helping people understand:

* How to stay safe on the internet
* How to protect personal data
* How to avoid cyber threats like scams, fraud, and hacking
* Best practices for using social media and online platforms

It is designed in a **simple and practical way**, so anyone can follow and apply it in real life.

## Key Topics Covered

### 1. Online Privacy

* Managing account privacy settings
* Reducing personal data exposure
* Controlling activity tracking

### 2. Account Security

* Using strong passwords
* Enabling 2FA (Two-Factor Authentication)
* Clearing old login sessions

### 3. Social Media Safety

* Limiting profile visibility
* Avoiding oversharing
* Managing permissions

### 4. Messaging Apps Safety

* Disabling unnecessary features
* Using disappearing messages
* Avoiding payments via chat apps

### 5. Google & Data Management

* Managing activity controls
* Deleting unnecessary data
* Reviewing connected devices

Being digitally aware = being **safe, smart, and secure online**
